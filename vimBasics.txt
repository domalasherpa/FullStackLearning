Day 1: Coventional Commit, Vim and Git

Conventional Commit:


Vim

Mode Switching
Esc	Enter a normal mode
i	Enter a insert mode
:	Enter command mode
R	Enter replace mode
v	Enter visual mode
V	Enter visual line mode

Commands

:set number	Show number in the vim
:w		Save the current file
:wq		Save the current file and close it
:w newname	Save current copy as a newname but continue editing the original
:x		Write any changes to the file and close the file
